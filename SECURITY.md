# Security Policy

## Supported Versions

agent-shield is pre-1.0. Only the most recent tagged release receives security
fixes. Older tags are frozen reference points and will not be patched.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

Please report security issues privately via GitHub's security advisory flow:

  https://github.com/cschuman/agent-shield/security/advisories/new

Do **not** open a public issue for vulnerabilities. Expect an initial response
within 7 days. Coordinated disclosure timelines are negotiated case-by-case.

For non-security bugs, open a regular issue.
